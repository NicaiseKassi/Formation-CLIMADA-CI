🔍 POURQUOI BASH NE MARCHE PAS SUR WINDOWS ?

Bash est un shell Unix/Linux. Windows utilise :
  • CMD (Invite de commandes)
  • PowerShell
  • Anaconda Prompt

Les fichiers .sh (scripts bash) ne fonctionnent pas sur Windows.
Les fichiers .bat (scripts batch) fonctionnent sur Windows.
